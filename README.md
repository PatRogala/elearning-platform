## Setup

1. Pull down the app from version control
2. `docker-compose up`
3. `docker-compose run --rm web bin/setup`

## Running The App

1. `docker-compose up`

## Tests and CI

1. `docker-compose run --rm web bin/ci`

## Production

* All runtime configuration should be supplied
in the UNIX environment
